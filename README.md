# Random Cats

Додаток, який показує випадкових милих котиків.  

## Функціонал

- Написано на класових компонентах React;
- Викориснано бібліотеку JS Axios;
- Отримання рандомних зображень котів з API  
  `https://cataas.com/api/cats?tags=cute`
- Анімація появи сердечок при зміні котика
- Кнопка для генерації нового котика

## Технології

- React (Class Components)
- Axios
- JavaScript
- CSS3
- Vite
